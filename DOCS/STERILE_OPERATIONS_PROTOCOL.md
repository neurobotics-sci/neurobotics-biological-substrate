### Bubo Sterile Operations Protocol v1.0
**Scope**: This repository is a clinical neuromorphic research platform.
**Hardening**: All commits must be hardware-signed via YubiKey FIDO2 (ECDSA-SK).
**Boundary**: The *Bubo sapiens* lineage is explicitly excluded from this substrate.
**Verification**: Any commit lacking a cryptographic signature is a breach of integrity.
